Edge detection algorithm on DSP

1)-Description

Edge detection is an image processing algorithm used to locate points in an image who correspond to a sudden change in the light intensity. These changes generally reflect important events or changes in world properties. We implemented this amazing algorithm on a DSP card using the C programming language, then we applied advanced assembler techniques to increase the processing power and give the system more performance

2)-Tools

  • Texas Instruments- TMS320C6000 DSK DSP card
  • Languages: C , Assembler
  • CCS CodeComposerStudio environment
